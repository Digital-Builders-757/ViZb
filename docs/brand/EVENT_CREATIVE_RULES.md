# Event creative rules — ViBE

**Last updated:** March 23, 2026

Rules for **flyers, promo assets, and event presentation** in-product and in marketing.

## Flyers (product)

- **Hero asset** for discovery; respect aspect ratio and file constraints from upload UI + storage policy.
- **Alt text / accessibility:** meaningful description, not “image1”.
- **Credit:** photographer/artist when provided (`BRAND_SYSTEM` imagery rules).

## Titles and slugs

- Titles are **human-facing truth**; slugs stable for sharing — changing slug breaks links; prefer rare changes with redirects if ever added.

## Status badges

- Use **mono uppercase label** pattern from `BRAND_SYSTEM` dashboard patterns — pending / live / ended / sold out.

## Brand check

Any new event UI block → **`/brand-check`** + **`/event-flow`** if lifecycle or RSVP/check-in touched.
