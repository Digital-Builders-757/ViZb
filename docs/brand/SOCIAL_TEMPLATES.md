# Social templates — ViBE

**Last updated:** March 23, 2026

Patterns for **organic social** announcing events and culture. Must match **voice** (`VOICE_AND_MESSAGING.md`) and **visual** (flyer-forward where possible).

## Event announcement (pattern)

1. **Hook** — date + city + vibe (one line).  
2. **Flyer or key visual** — credit photographer if known.  
3. **CTA** — link to ViBE event page (canonical URL).  
4. **Hashtags** — light, scene-specific; avoid clutter.

## Do / don’t

- **Do** reuse official event title spelling from the product.  
- **Don’t** invent ticket prices or lineups not in the product DB.
